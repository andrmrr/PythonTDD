This is a project created following the book "Test-Driven Development with Python" by Harry J.W. Percival.
The project is a simple web application build using Django framework to showcase different testing scenarios. 
It is tested and deployed using Docker and Ansible as well as CI on GitLab.
For the server provisioning DigitalOcean was used and the domain name kalekos.xyz was found on godaddy.com
Some of the configuration files are not uploaded, but saved only locally.
